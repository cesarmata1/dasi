# DASI

Este repositorio contiene el código desarrollado para la asignatura **DASI** (Desarrollo de Aplicaciones Software e Ingeniería). El proyecto reúne ejercicios y prácticas orientados al aprendizaje de conceptos de desarrollo de software aplicados durante el curso.

## Descripción

El contenido del repositorio incluye diferentes actividades realizadas en la asignatura, organizadas de forma que reflejan la progresión de aprendizaje en el desarrollo de soluciones usando **Java**. El proyecto está estructurado para facilitar su compilación y ejecución desde un entorno de desarrollo.

## Tecnologías utilizadas

- Java
- Entorno de desarrollo compatible con Java (por ejemplo, Eclipse, NetBeans o IntelliJ IDEA)
